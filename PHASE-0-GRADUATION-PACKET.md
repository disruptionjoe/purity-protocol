## Claim Discipline

This packet uses four labels:

- **Established here:** directly supported by current ratified repository
  authority or current-state evidence within the cited scope.
- **Being tested:** an active bounded inquiry with a stated method. No item is
  assigned this label merely because it appears on the research agenda.
- **Hypothesized:** a plausible founding idea without sufficient evidence.
- **Unknown:** no current justified answer or decision.

Compilation is not validation. A cited rule proves what the repository
currently governs; it does not prove that a future mechanism, transfer, or
institution will work.

## First Eligibility-Gate Work Still Required

A decision-ready gate must satisfy all eight requirements in
`GOVERNANCE.md#Eligibility changes`. The current evidence specification selects
authoritative project effort only as an analytical class and defines a
hypothetical bounded-research role so that evidence needs can be tested. It
does not define an eligible role or a gate.

Before a concrete gate is proposed, Lane 1 should at minimum:

1. choose which one of the three eligibility classes is being considered;
2. define the bounded role and prohibited consequences of that expansion;
3. state candidate qualification and evidence standards;
4. define consent, exit, review, appeal, and attributable-creation treatment;
5. specify capture, collusion, exclusion, and epistemic-integrity threats;
6. name measurable leading and stop indicators;
7. separate every relevant dimension of reversibility; and
8. preserve a separate Joe activation decision after any gate is judged
   satisfied.

This is a work specification, not a gate and not an eligibility proposal.
